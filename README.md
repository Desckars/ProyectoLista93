El objetivo de este proyecto es diseñarle a La Vanguardia Del Ceibo una pagina web que los ayude a llegar a nuevas personas que contengan el mismo ideal politico.

Para ello utilizamos un proveedor externo como Supabase para el almacenamiento de fotos y texto, por otro lado usamos Netlify como nuestro proveedor de servidor para tener la web disponible 24/7(Siempre y cuando la pagina tenga actividad, en caso contrario ambos proveedores para ahorrar recursos ponen el proyecto en pausa)

El Proyecto cuenta con la siguiente distribucion:

Prototipo1 ---> Carpeta Raiz del proyecto, contiene el README y los .html
|---> css ---> Contiene los estilos de la pagina
|---> Imagenes ---> Contiene logos necesario para el correcto funcionamiento de la pagina
|---> js ---> Contiene archivos JS con la logica 

Link a la pagina ---> https://vanguardia-del-ceibo.netlify.app/index.html
